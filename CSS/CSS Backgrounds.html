CSS Backgrounds & Opacity / Transparency

1. Background Color

background-color: Sets the background color of an element.
background-color: #3498db;

2. Background Image

background-image: Sets an image as the background.
background-image: url('image.jpg');
background-size: Controls image scaling.
background-repeat: Controls repetition.
background-position: Sets image position.

3. Background Gradient

background: Can use gradients.
background: linear-gradient(135deg, #e66465, #9198e5);

4. Multiple Backgrounds

Multiple backgrounds: You can layer images and gradients.
background: url('image.jpg') left center/100px no-repeat,
  linear-gradient(90deg, #f6d365 0%, #fd6e6a 100%);

5. Transparent Background

background: transparent; Makes the background see-through.
background: transparent;

6. Opacity Property

opacity: Sets the transparency of the entire element (including children).
opacity: 0.6;
Child elements are also affected!

Note: opacity affects the whole element and its content.

7. RGBA for Background Transparency

rgba: Use RGBA colors for background transparency.
background: rgba(231, 76, 60, 0.5);

Note: Only the background is transparent, not the content.

8. Useful Background Properties

  • background-attachment: fixed or scroll
  • background-clip: Controls painting area (border-box, padding-box, content-box)
  • background-origin: Sets background positioning area
  • background-blend-mode: Blends background layers
  • background-repeat: repeat, no-repeat, repeat-x, repeat-y
  • background-size: cover, contain, or custom size

Summary

  • Use background-color, background-image, background for backgrounds.
  • Use opacity for full element transparency.
  • Use rgba() or hsla() for background-only transparency.
  • Combine multiple backgrounds and gradients for advanced effects.
