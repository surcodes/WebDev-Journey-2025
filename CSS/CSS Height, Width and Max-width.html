CSS Height, Width, and Max-width

The height and width properties set the size of an element.
The max-width property restricts the maximum width of an element.

Value types:

  • auto - Default. The browser calculates the height and width.
  • length - Defines the height or width in px, cm, em, etc.
  • % - Defines the height or width in percent of the containing block.
  • initial - Sets the height or width to its default value.
  • inherit - The height or width will be inherited from its parent value.

Length Units: Height and width can use px, pt, cm, etc.
height: 100px; width: 300px;
Percentage: Height and width as a percentage of the containing element's height and width.
height: 50%; width: 50%;
Parent Element:
height: 150px; width: 400px;
inherit: Height and width are inherited from the parent element.
height: inherit; width: inherit;
max-width: Restricts the maximum width of an element.
width: 100%; max-width: 500px;
Resize the browser window to see the effect.
auto: Browser calculates the height and width.
width: auto; height: auto;
initial: Sets the height and width to their default values.
width: initial; height: initial;

Note

When using percentage values for height, the parent element must have a defined height. Otherwise, the percentage height will not work as expected.
max-width is useful for making layouts responsive, preventing elements from becoming too wide.
