CSS Padding

The CSS padding properties are used to generate space around an element's content, inside of any defined borders.
Padding creates space between the content and the border of an element.

Padding - Individual Sides

padding-top: Sets the top padding of an element.
padding-top: 40px;
padding-right: Sets the right padding of an element.
padding-right: 40px;
padding-bottom: Sets the bottom padding of an element.
padding-bottom: 40px;
padding-left: Sets the left padding of an element.
padding-left: 40px;

Padding - Shorthand Property

padding: Shorthand for setting all four paddings in one declaration.
padding: 20px 40px 10px 60px; (top right bottom left)

Padding Value Types

Percentage: Padding as a percentage of the containing element's width.
padding-left: 10%;
Length Units: Padding can use px, pt, cm, etc.
padding: 30pt;
inherit: Padding is inherited from the parent element.
padding: inherit;

Note

Negative values are not allowed for padding.
