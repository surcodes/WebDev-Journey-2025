CSS Selectors Explained

CSS id Selector

The id selector uses #idname to select a single, unique element.
Example: #unique { color: green; }

CSS class Selector

The class selector uses .classname to select all elements with that class.
Example: .highlight { background: yellow; }

CSS Universal Selector

The universal selector uses * to select all elements on the page.
Example: * { font-family: Arial, sans-serif; }

CSS Grouping Selector

Grouping Selector Example

This paragraph uses the .grouped class.

This paragraph uses the #grouped-id id.

The grouping selector lets you apply the same styles to multiple selectors by separating them with commas.
Example: h3, .grouped, #grouped-id { background: #f0f0f0; color: navy; }
