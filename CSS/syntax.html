CSS Syntax and Usage

CSS (Cascading Style Sheets) is used to style HTML elements. The basic syntax of CSS consists of a selector and a declaration block.

CSS Syntax

selector {
    property: value;
    property2: value2;
}
  • Selector: Specifies which HTML elements to style (e.g., p, .class, #id).
  • Property: The style attribute you want to change (e.g., color, font-size).
  • Value: The value for the property (e.g., red, 16px).

Example

p {
    color: blue;
    font-size: 18px;
}

This example styles all <p> elements to have blue text and a font size of 18 pixels.

How to Use CSS

  1. Inline CSS: Add the style attribute directly to an HTML element.
    <p style="color: red;">This is red text.</p>
  2. Internal CSS: Place CSS rules inside a <style> tag in the HTML <head>.
    <style>
    h1 { color: green; }
</style>
  3. External CSS: Write CSS in a separate .css file and link it with <link>.
    <link rel="stylesheet" href="styles.css">
